== Description ==

Allows you to add your site verification tag to your site.

The plugin will generate the site verification meta tag used to validate a site on [siteby.com](https://siteby.com/).

SiteBy is a free crowdsourced directory of websites and the people who built them. A first-of-it's-kind, site-centric directory of web professional talent.

[Set up your profile and add your sites today!](https://siteby.com/)

== Installation ==

1. Use the Add New Plugin in the WordPress admin area
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can find the settings and documentation under Settings -> SiteBy Verification

Then to make plugin work on pages - make sure the wp_head() function is utilized in the header.

== Frequently Asked Questions ==
 
= Does the plugin automatically add my site to SiteBy? =
 
You need to create an account at [SiteBy](https://siteby.com/) and then follow the installation instructions.
